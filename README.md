# Android app to control Robot car via Bluetooth module
## Equipment
* L298N DC motor driver
* Arduino UNO
* Sensor shield V5
* Ultrasonic sensor
* Bluetooth module
* DC Motor
* Servo motor
* Line sensor
* Tachometer
## Pin assignment
* A0 Ultrasonic sensor
* A1 Ultrasonic sensor
* A2 Line sensor 1
* A3 Line sensor 2
* A4 Line sensor 3
* A5 Line sensor 4
* D0 Bluetooth TX
* D1 Bluetooth RX
* D2 Tachometer 1
* D3 Tachometer 2
* D5 Servo motor
* D6 DC motor 1
* D7 DC motor 1
* D8 DC motor 1
* D9 DC motor 2
* D10 DC motor 2
* D11 DC motor 2
## Function
* Line tracking
* Object evasion
* Move zigzag
* Distance measurement
* Manual movement
* Moving by data in record.txt
* Select robot type via mac address
